# Contributing

Thanks for helping improve `aiscope`.

## Install

```bash
git clone https://github.com/YOUR_USERNAME/aiscope.git
cd aiscope
npm install
```

## Run Locally

```bash
npm link
aiscope help
```

## Development

```bash
npm test
npm run lint
```

Keep code readable, small, and focused. Prefer Node.js built-ins unless a dependency clearly improves the project.

## Issues

Open an issue with:

- what you expected
- what happened
- your OS and shell
- the relevant command output with secrets removed

## Pull Requests

Before submitting a PR:

- run tests
- update docs if behavior changes
- avoid committing real env files or secrets
- keep unrelated refactors out of the PR
