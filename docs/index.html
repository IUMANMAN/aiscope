Auto-loading local scopes for AI coding agents.

Stop exporting global API keys. Stop typing wrapper commands. Enter a project folder, run codex or claude, and the right environment is already active.

No wrapper commands

Run Codex, Claude Code, npm, Python, Wrangler, Vercel, or any CLI exactly as you normally would.

No global secrets

Keep project secrets in a local vault and avoid exporting every API key in your shell config.

Built for AI workflows

Use project and skill scopes today, with agents, workflows, and encrypted vaults on the roadmap.

Works with any CLI

The shell hook exports normal environment variables, so existing tools do not need integrations.
