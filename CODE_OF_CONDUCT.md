# Code of Conduct

This project is for builders who want safer local AI development workflows.

Please keep discussions respectful, specific, and useful:

- Be kind and direct.
- Assume good intent, but clarify when something is ambiguous.
- Do not post secrets, API keys, tokens, private env files, or screenshots containing secrets.
- Keep issues focused on reproducible behavior, clear requests, or concrete documentation gaps.
- Avoid harassment, personal attacks, spam, and unrelated promotion.

Maintainers may edit, hide, or remove content that makes the project harder to use or maintain.
